# L3 Summaries

実施したことの概略を置く。

## 置くもの

- タスク完了後の短い要約。
- 次回AIが思い出すためのリンク集。
- 詳細正本への入口。

## 置かないもの

- 詳細な作業ログ。
- 長い引用。
- `tasks/` や `knowledge/` と同じ本文。

## 書き方

- 1件1ファイルを基本にする。
- 詳細は `tasks/`, `knowledge/`, `sources/` へリンクする。
- 次回の想起トリガーを1から3個だけ書く。
