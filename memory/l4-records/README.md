# L4 Records

詳細記録、または詳細正本への入口を置く。

## 置くもの

- 長くなった検討の入口。
- 複数ファイルにまたがる作業の案内。
- 詳細正本が外にある場合のリンクハブ。

## 置かないもの

- `tasks/` に置くべき実施履歴の重複。
- `sources/` に置くべき根拠資料の重複。
- `knowledge/` に置くべき整理済み知識の重複。

## 原則

L4 は詳細を無制限に増やす場所ではない。詳細正本が別にあるなら、入口とリンクだけにする。
