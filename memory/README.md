# Memory Layer

LLM が効率よく思い出すための記憶階層。

## 層

| 層 | Path | 役割 |
|---|---|---|
| L1 | `l1-triggers.md` | 常時読む短いキーワードとリンク |
| L2 | `l2-models/` | 未確定の理解モデル、仮説、違和感 |
| L3 | `l3-summaries/` | 実施したことの概略 |
| L4 | `l4-records/` | 詳細記録または詳細正本への入口 |

## 原則

- L1 は本文を書かず、短いトリガーとリンクだけにする。
- L2 は未確定として扱い、決定事項を書かない。
- L3 は概要だけにし、詳細は `tasks/` や `knowledge/` にリンクする。
- L4 は詳細を書くか、詳細正本への入口にする。
- 同じ内容を各層へコピーしない。
