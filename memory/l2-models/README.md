# L2 Models

未確定の理解モデル、仮説、違和感を置く。

## 置くもの

- まだ正本化しない理解モデル。
- 複数案が残っている分類、構造、運用案。
- 後で検証したい違和感や仮説。

## 置かないもの

- 決定事項。
- 根拠本文。
- 作業履歴の詳細。

## 昇格先

- 決定したら `knowledge/decisions/`。
- 実施したら `tasks/`。
- 根拠が必要なら `sources/`。
